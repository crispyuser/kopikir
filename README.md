# Warung Kopikir - Coffee Shop Website

A responsive coffee shop website that can be deployed to GitHub Pages. This version uses static HTML, CSS, and JavaScript instead of PHP to work with GitHub Pages' static hosting.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Interactive elements with JavaScript
- Contact form with validation
- Modern UI with Tailwind CSS

## Files

- `index.html` - Main landing page
- `menu.html` - Menu listing page
- `testimonials.html` - Customer reviews page
- `contact.html` - Contact page with form
- `assets/css/custom.css` - Custom styling
- `assets/js/main.js` - JavaScript for interactions

## Deployment to GitHub Pages

1. Create a new repository on GitHub
2. Push all files to the repository
3. Go to repository Settings
4. Scroll down to "Pages" section
5. Select "Deploy from a branch"
6. Choose "main" branch and "/ (root)" folder
7. Click "Save"
8. Your site will be available at `https://[username].github.io/[repository-name]/`

## Local Development

To run the site locally:
1. Open `index.html` in a web browser
2. Or use a local server like Live Server extension in VS Code

## Notes

- This version uses static data instead of a database
- Contact form doesn't actually send emails (would require a backend service)
- All dynamic PHP functionality has been converted to static HTML with JavaScript

## Customization

To customize the content:
1. Edit the HTML files directly
2. Modify the static data in the JavaScript sections
3. Update the Tailwind CSS classes for styling changes
